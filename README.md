#Project description
This is my very first CRUD app. I built this app as a way to practice how to use Flask and finally apply python logic in my code. This app Creates a new contact, Updates/edits the contact  and deletes the contact whenever necessary.

#Features
 - Responsive HTML templates using Jinja2

 - SQLite database handled through SQLAlchemy

 - Form validation for required fields

 - Clean project structure (templates/, static/)

#Tech stack
 - Python 3.x
 - Flask
 - SQLAlchemy
 - SQLite
 - HTML/CSS
